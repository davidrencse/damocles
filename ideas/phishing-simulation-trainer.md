# Phishing Simulation Trainer

## Summary

 A web application designed to simulate phishing attacks to educate users on recognizing and avoiding these threats. Users can experience realistic phishing scenarios and receive feedback on their responses.

## Category

 Security Awareness

## Difficulty

 Intermediate

## Safe Scope

 Focus on providing educational simulations without collecting any user data or personal information.

## Features

 - User registration and authentication
- Phishing simulation scenarios with different styles (emails, websites)
- Feedback and scoring system after each simulation
- Admin interface for creating and managing phishing scenarios
- Progress tracking and analytics for users

## Tech Stack

 - React
- TypeScript
- Vite
- Tailwind
- Node.js
- Express

## File Structure

 - src/
- src/components/
- src/pages/
- src/hooks/
- src/utils/
- src/styles/
- public/

## Acceptance Criteria

 - Users can register and log in to the application.
- Users can participate in multiple phishing simulations.
- Feedback is provided for each simulation based on user choices.
- Admin users can create and modify phishing scenarios.
- Progress tracking is visible to users.

## Documentation Requirements

 - User manual explaining how to register and use the application.
- API documentation for the backend server.
- Contributing guidelines for open-source contributions.
- Installation instructions for local development.
