# Secure Chat Application

## Summary

A real-time chat application that implements end-to-end encryption and secure authentication mechanisms, showcasing best practices in security and modern web development.

## Category

Cybersecurity & Web Development

## Difficulty

Intermediate

## Safe Scope

The project focuses on front-end development while simulating backend security features with fake APIs to demonstrate functionality without risking actual data.

## Features

- User registration and login with JWT authentication
- Real-time chat functionality using WebSockets
- End-to-end encryption for messages
- User profile management
- Responsive design with Tailwind CSS

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Socket.io
- dotenv for environment variables

## File Structure

- src/
- src/components/
- src/pages/
- src/styles/
- public/
- package.json
- vite.config.ts

## Acceptance Criteria

- Users must be able to register and log in securely.
- Real-time messaging between users is operational.
- Messages should be encrypted before sending and decrypted upon receipt.
- The application is fully responsive and adheres to accessibility standards.
