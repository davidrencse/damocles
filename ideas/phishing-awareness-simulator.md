# Phishing Awareness Simulator

## Summary

 A simulated phishing environment designed to educate users on identifying phishing attempts through interactive scenarios and quizzes.

## Category

 Security Education

## Difficulty

 Medium

## Safe Scope

 The project will strictly focus on simulating phishing scenarios and educating users, avoiding any actual malicious functionality.

## Features

 - Interactive phishing email examples
- Quizzes to test user knowledge after each scenario
- Dashboard for tracking user progress
- Resource section with links to anti-phishing tools and guidelines
- Mobile-friendly interface using Tailwind

## Tech Stack

 - React
- TypeScript
- Vite
- Tailwind

## File Structure

 - src/
- src/components/
- src/pages/
- src/hooks/
- src/assets/
- src/styles/
- public/

## Acceptance Criteria

 - Users can navigate through different phishing scenarios
- Quizzes reflect user understanding and adapt to previous answers
- Progress tracking is visually represented in the dashboard
