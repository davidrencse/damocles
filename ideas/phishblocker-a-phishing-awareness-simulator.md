# PhishBlocker: A Phishing Awareness Simulator

## Summary

 PhishBlocker is an interactive web application designed to educate users about phishing attacks through simulated email scenarios. Users can identify phishing attempts and learn about the red flags that indicate fraudulent messages.

## Category

 Cybersecurity Education

## Difficulty

 Intermediate

## Safe Scope

 The project will allow users to engage with simulated phishing emails without any real threat. It includes a scoring system based on user responses to raise awareness about phishing tactics.

## Features

 - User registration and login
- A library of simulated phishing emails
- Interactive email inspection with drag-and-drop features
- Scoring system to track user performance
- Educational resources on how to identify phishing emails
- Feedback on user choices with explanations of phishing tactics

## Acceptance Criteria

 - User can register and log in successfully.
- User can view, inspect, and respond to simulated phishing emails.
- Scoring system calculates and displays performance metrics accurately.
- Users receive feedback on each choice with educational explanations.
- All features should be responsive and functional across various devices.
